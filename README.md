# Simple Scala Service  

Simple HTTP Service with tests.

### Prerequisites

Scala Version 2.12.6

SBT Version 1.1.6

### Installing

Compile using SBT: 

```
sbt compile
```

## Running the tests

Test using SBT:

```
sbt test
```
